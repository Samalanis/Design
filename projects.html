A short quiz showcasing CSS, HTML, and Javascript. The project touches on the basics of using functions to hide divs, show divs, return items from an array and visually appeal to the user. The project required a very minimal usage of HTML to create a clean enviorment. The majority of the code was place within the script.js where I relied heavily on the ability to call on other functions within the script. All in all it's a fun way to kill time and test your skills against a self proclaimed pro bass fisherman such as myself.
